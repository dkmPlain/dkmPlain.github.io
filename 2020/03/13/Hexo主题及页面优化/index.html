Hexo主题及页面优化

一、常用的主题

  • NexT: https://github.com/iissnan/hexo-theme-next

  • Yilia: https://github.com/litten/hexo-theme-yilia，这里有一个基于Yilia的blog：https://elevenzou.github.io。

  • Yelee: https://github.com/MOxFIVE/hexo-theme-yelee ，双栏风格，由Yilia优化而来，引用Yelee作者的介绍”Yelee —— 简而不减 Hexo 双栏博客主题”。

  • Maupassant: https://github.com/tufu9441/maupassant-hexo ，属于简约风格类型。

二、安装

  1. Win+R cmd 命令行，网站目录下使用git clone从远程拷贝项目。

    git clone https://github.com/MOxFIVE/hexo-theme-yelee themes/yelee

  2. 修改网站目录下的_config.yml （站点配置文件）文件。

    theme: yelee

  3. 重新部署，输入命令：

    • hexo clean
    • hexo g
    • hexo d

  4. 更新主题(一般不要更新)，主题目录themes/yelee下：

    git pull

三、基础设置

  1. 语言切换，_config.yml 站点文件：

    language: zh-Hans

  2. 个人头像，默认头像存储于 yelee/source/img/avatar.png，_config.yml 主题文件修改：

    avatar: /img/xxx.png （自定义）

  3. 文章摘要，内容区适当位置使用 +< !–more–> 标识，或者使用 +description: “Welcome to …” 放置于Front-matter 区域内（仅限于文字描述）。

  4. 评论设置，Disqus，duoshuo及youyan评论，_config.yml 主题文件选择一项on: ture，其余#注释掉。

    参考：http://moxfive.coding.me/yelee/2.Basic-Usage/comment.html

  5. 主菜单，边栏具体项目可选，_config.yml 主题文件 menu项。

  6. 标签云页面，网站根目录下使用 Hexo 命令新建一个名为 tags 的页面：

    ~\web-host>hexo new page tags

    在source目录下生成tags文件夹，创建index.md文件，修改文件内的title项。

  7. 关于我页面，网站根目录下使用 Hexo 命令新建一个名为about 的页面：

    ~\web-host>hexo new page about

    在source目录下生成about文件夹，创建index.md文件。

  8. 本地搜索，需要安装插件hexo-generator-search

    配置在_config.yml 主题文件 search项，默认#on: true （主页加载动画时会出现javascript错误），去掉注释，启用search项，即改为on: true 。

  9. 浏览器Tab小图标，位置：yelee/source/favicon.png，配置在_config.yml 主题文件 favicon项。

  10. 若网站放在子目录：

    _config.yml 站点文件root项，非子目录”/“。

    _config.yml 主题文件root_url项，非子目录留空。

  11. 社交图标，_config.yml 主题文件subnav项。

    详细：http://moxfive.coding.me/yelee/2.Basic-Usage/social-bar.html

  12. 无效网址404页面，网站根目录下使用 Hexo 命令新建一个名为404 的页面

    ~\web-host>hexo new page 404

    在source目录下生成404文件夹，文件夹下的index.md 文件中添加 permalink: /404。

  13. 修改网站成立年份，_config.yml 主题文件 since项。

四、样式自定义

  1. 背景图片，/yelee/source/background/，配置在_config.yml 主题文件background_image项。
  2. 代码高亮和文本高亮，配置在_config.yml 主题文件code_block项和inline_code项。
  3. 字号调节，配置在_config.yml 主题文件base_font_size项。
  4. 引用区域，配置在_config.yml 主题文件blockquote_style项。
  5. 标题风格，配置在_config.yml 主题文件heading_style项，默认0，改为2。
  6. 列表风格，配置在_config.yml 主题文件list_style项，默认0（兼容性差，多行显示会引起换行），改为1。
  7. 主题配色方案，配置在_config.yml 主题文件color_scheme项。
  8. 左边栏宽度，配置在_config.yml 主题文件left_col_width项。
  9. 页面进度条，配置在_config.yml 主题文件progressBar项，默认开启，选择关闭。
  10. 提示框美化，配置在_config.yml 主题文件jquery_ui项。

五、页面设置

  1. 文章目录，是否启用文章目录，配置在_config.yml 主题文件toc项。
  2. 版权信息，配置在_config.yml 主题文件copyright项。
  3. 边栏鸟屋/友情链接/关于我，配置在配置在_config.yml 主题文件tagcloud、friends、aboutme项。
  4. 新窗口打开方式，配置在配置在_config.yml 主题文件open_in_new项。
  5. 动画效果，配置在配置在_config.yml 主题文件animate项。
  6. 标签标题切换，配置在配置在tab_title_change项。
